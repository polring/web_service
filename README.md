# 미니 포트폴리오 프로젝트

자유 주제로 시작한 개인 미니 포트폴리오입니다. 간단한 디자인을 유지하면서도 최소 3개의 페이지(기능)를 포함하도록 구성했습니다.

## 브라우저에서 확인
`http://127.0.0.1:5000`


## 요구 조건 충족
- 자유 주제: 개인 포트폴리오 소개
- 간단한 디자인: 단일 레이아웃과 가벼운 스타일 사용
- 3개 이상 페이지: `소개`, `기술`, `프로젝트`, `연락`

## 주요 페이지
- `/`: 소개 및 프로젝트 개요
- `/skills`: 기술 스택과 관심 분야
- `/projects`: 미니 프로젝트 아카이브
- `/contact`: 연락 및 목표

## 구성
- `app.py`: 앱 팩토리와 블루프린트 등록
- `blueprints/`: 라우트 모음
- `templates/`: Jinja 템플릿
- `data/`: 페이지 데이터(JSON)
- `requirements.txt`: 기능 요구사항 목록
- `plugins/`: 플러그인 폴더

## 향후 아이디어
- 요구사항 목록은 `requirements.txt`에 정리되어 있습니다.
