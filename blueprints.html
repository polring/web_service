blueprints package

Submodules

blueprints.api module

JSON API — 외부에서 포트폴리오 데이터를 조회.

OpenAPI 조각은 swagger/*.yml 에 두고, Flasgger swag_from 으로 연결합니다. Sphinx autodoc 과 docstring 형식이 충돌하지 않도록 스펙은 파일로 분리했습니다.

blueprints.api.get_note(item_id)

경로의 item_id 에 해당하는 노트 한 건을 반환합니다. 없으면 404.

blueprints.api.get_notes()

노트 목록 JSON 배열을 반환합니다.

blueprints.api.get_profile()

프로필 JSON을 반환합니다 (profile.json).

blueprints.api.get_project(item_id)

경로의 item_id 에 해당하는 프로젝트 한 건을 반환합니다. 없으면 404.

blueprints.api.get_projects()

프로젝트 목록 JSON 배열을 반환합니다.

포트폴리오 화면을 app/data/*.json 과 app/templates 로 구성합니다.
