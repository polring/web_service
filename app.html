app 모듈 (애플리케이션 팩토리)

Flask 앱 생성, 블루프린트 등록, 플러그인 로드, Flasgger(Swagger UI) 초기화를 담당합니다.

Flask 애플리케이션 진입점 및 팩토리.

create_app() 으로 앱 인스턴스를 만들고, main / api 블루프린트와 선택 플러그인, Flasgger(Swagger UI)를 등록합니다.

app.create_app() → Flask[source]

Flask 앱을 생성하고 확장(블루프린트·플러그인·Flasgger)을 등록합니다.

Returns:

설정이 끝난 Flask 인스턴스.

app.load_plugins(app: Flask) → None[source]

app/plugins 패키지 아래 모듈을 동적으로 로드합니다.

각 모듈에 register(app) 이 있으면 호출하고, 없고 bp 가 있으면 블루프린트로 등록합니다. PLUGINS=a,b 로 로드할 모듈 이름을 제한할 수 있습니다.

Parameters:

app – Flask 애플리케이션 인스턴스.
