Flask 포트폴리오 & JSON API
============================

**Flask** 기반 개인 포트폴리오 웹앱과 **JSON 조회 API**를 제공합니다. 데이터는 ``app/data/*.json`` 에 두고,
Jinja2 템플릿으로 페이지를 렌더링합니다. API는 **Flasgger** 로 OpenAPI(Swagger UI) 문서를 노출합니다.

.. note::

   이 사이트는 **Sphinx autodoc** 으로 Python 소스의 **docstring** 에서 생성되었습니다.
   배포: `GitHub Pages <https://polring.github.io/web_service/>`__ (``gh-pages`` 브랜치).

주요 기능
---------

* **페이지**: 홈, 기술, 프로젝트, 연락, 소개 (``blueprints.main``)
* **API**: ``/api/profile``, ``/api/projects``, ``/api/projects/<id>``, ``/api/notes``, … (``blueprints.api``)
* **데이터 접근**: ``services.json_store`` — JSON 파일 로드 및 id 검색
* **플러그인**: ``app/plugins`` 디렉터리 동적 로드 (``PLUGINS`` 환경 변수)
* **API 문서(UI)**: 로컬 실행 시 ``/apidocs/`` (Flasgger)
